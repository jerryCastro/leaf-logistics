## Getting Started

### Prerequisites

* npm

### Installation

1. Get a free API Key at [https://newsapi.org/](https://newsapi.org/)
2. Clone the repo
   ```sh
   git clone https://github.com/jerryCastro/leaf-logistics.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your API Key in `.env`
   ```sh
   cp .env.example .env
   ```
   ```META
   API_KEY=YOUR_API_KEY
   ```

### Run
   ```sh
   npm run dev
   ```
